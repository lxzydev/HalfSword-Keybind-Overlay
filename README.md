The overlay reads your key binds directly from the game at runtime.
Toggle on/off with F6

Configuration
Open "main.lua" in a text editor. The config options are at the top of the file:
TOGGLE_KEY - Key to show/hide the overlay (default: F6)
SHOW_ON_START - Whether the overlay is visible when the game starts (default: true)
OVERLAY_POSITION - Where on screen to place it (default: topleft)
FONT_SIZE -Text size (default: 12)
BG_OPACITY - Background darkness, 0.0 = transparent, 1.0 = solid black (default: 0.0)

Requirements
UE4SS v3.0.1 or newer

Installation
Download
Extract to "Half Sword\HalfswordUE5\Binaries\Win64\Half Sword\HalfswordUE5\Binaries\Win64\Half Sword\HalfswordUE5\Binaries\Win64\"
You should have "Win64\dwmapi.dll and a Win64\ue4ss\" folder

2. Install KeybindOverlay
Copy the "KeybindOverlay" folder into "Win64\ue4ss\Mods\Win64\ue4ss\Mods\"

3. Launch the game
The overlay should appear automatically
Press F6 to toggle it on/off
